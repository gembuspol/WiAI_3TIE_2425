Styl lokalny

  • <body style="color:white"> styl lokalny wewnątrz znacznika
  • <span style="color:white">...</span> styl lokalny jako rozciągnięty blok - może być na kilku znacznikach
  • <section style="color:white">...</section> styl lokalny jako blok

Wewnętrzny arkusz styli

  • <style></style> styl wewnętrzny - wpisujemy go ew nagłówku strony - sekcji head

Zewnętrzny arkusz styli

  • <link rel="stylesheet" type="txt/css" href="arkusz.css"> styl zewewnętrzny zapisujemy go w osobnym pliku

import arkusza stylów

  • @import urt("..."); - import arkusza z zewnętrznego źródła

Kaskadowość arkuszy stylów

Ważnaść poszczególnych rodzajów stylów(numer 1 ma najwyższy piorytet - pierwszeństwo w wyświetlnaiu informacji na stronie, a styl nr 7 ma najniższy

  1. Styl lokalny
  2. Rozciagniecie stylu (span)
  3. Wydzielone bloki (section w nim wpisane)
  4. Wewnętrzny arkusz styli (to co jest między znacznikami style
  5. Zewnętrzny arkusz styli
  6. Import stylów do zewnętrznego arkusza
  7. atrybuty definiowane w dokumencie HTML

Zasady Kaskadowości można zmieniać, służy do tego polecenioe !important i umieszcza się je w deklaracji stylu

Dziedziczenie

Jeżeli styl będzie sie odnosił do kilku elementów np. znaczników bloku footer to jeżeli będziemy ustawiac dla niego jakieś atrybuty, to te atrybuty będą się pojawiać u wszystkich. Jeżeli jeden z tych elementów zmienimy z wykorzystaniem np. id to musimy pamiętać aby wszystkie atrybu zmienić, nie zmienione atrybuty zostana zaczerpnięte z poprzedniego stylu

Składnia języka CSS

Język CSS składa się z: selektora (np. paragraf, znacznik HTML, klasa, identyfikator), właściwość (np. color), wartość(wyrażona w odpowiednich jednostakch
